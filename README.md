# Get my videos

App for searching videos on YT

## Getting Started

This app was created with create-react-app, thus you can use react-scripts for mainating all processes.

## Running

### with npm

```
npm install
npm start

```

### with yarn

```
yarn install
yarn start

```


## Running the tests

### with npm

```
npm test

```

### with yarn

```
yarn test

```
